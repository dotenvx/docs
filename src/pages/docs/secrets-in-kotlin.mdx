## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```kotlin {{ title: 'index.kt' }}
fun main() {
  val hello = System.getenv("HELLO") ?: ""
  println("Hello $hello")
}
```

```sh
kotlinc index.kt -include-runtime -d index.jar

dotenvx run -- java -jar index.jar
```
