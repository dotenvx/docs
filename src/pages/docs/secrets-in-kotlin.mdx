# Encrypt secrets in Kotlin

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the dotenvx cli.

<CodeGroup>

```bash {{ title: "curl" }}
curl -sfS https://dotenvx.sh | sh
```

</CodeGroup>

## 2. Encrypt env file

Encrypt your `.env` file.

```sh {{ title: '$ terminal' }}
dotenvx encrypt
```

## 3. Inject secrets

Then inject your encrypted secrets at runtime with `dotenvx run`.

```kotlin {{ title: 'index.kt' }}
fun main() {
  val hello = System.getenv("HELLO") ?: ""
  println("Hello $hello")
}
```

```sh {{ title: '$ terminal' }}
kotlinc index.kt -include-runtime -d index.jar

dotenvx run -- java -jar index.jar
```
