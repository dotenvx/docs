# Encrypt secrets in Rust

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Install dotenvx](/docs/install)
* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the dotenvx cli.

<CodeGroup>

```bash {{ title: "curl" }}
curl -sfS https://dotenvx.sh | sh
```

</CodeGroup>

## 2. Encrypt env file

Encrypt your `.env` file.

```sh {{ title: '$ terminal' }}
dotenvx encrypt
```

## 3. Inject secrets

Then inject your encrypted secrets at runtime with `dotenvx run`.

```rust {{ title: 'src/main.rs' }}
use std::env;

fn main() {
    let hello = env::var("HELLO").unwrap_or_default();
    println!("HELLO: {}", hello);
}
```

```sh {{ title: '$ terminal' }}
dotenvx run -- cargo run
```
