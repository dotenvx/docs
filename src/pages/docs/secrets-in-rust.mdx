## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```rust {{ title: 'src/main.rs' }}
use std::env;

fn main() {
    let hello = env::var("HELLO").unwrap_or_default();
    println!("HELLO: {}", hello);
}
```

```sh
dotenvx run -- cargo run
```
