# dotenvx ext ls -f

<GuideHeading lead={description} />

```sh
$ touch .env
touch .env.production
mkdir -p apps/backend
touch apps/backend/.env
touch apps/backend/.env.prod

dotenvx ext ls -f '**/.env.prod*'
├─ .env.production
└─ apps
   └─ backend
      └─ .env.prod
```
