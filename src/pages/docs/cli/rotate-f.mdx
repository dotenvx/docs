# dotenvx rotate -f

<GuideHeading lead={description} />

```sh
$ echo "HELLO=World" > .env
echo "HELLO=Production" > .env.production

dotenvx encrypt -f .env.production
◈ encrypted (.env.production)
dotenvx rotate -f .env.production
⟳ rotated (.env.production)
```
