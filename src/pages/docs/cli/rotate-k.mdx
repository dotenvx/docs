# dotenvx rotate -k

<p className="lead">{description}</p>

```sh
$ echo "HELLO=World\nHOLA=Mundo" > .env
dotenvx encrypt
◈ encrypted (.env)
dotenvx rotate -k HELLO
⟳ rotated (.env)
```

Even specify a glob pattern.

```sh
$ echo "HELLO=World\nHOLA=Mundo" > .env
dotenvx encrypt
◈ encrypted (.env)
dotenvx rotate -k "HE*"
⟳ rotated (.env)
```
