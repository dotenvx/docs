# Armor Pull

<GuideHeading lead={description} />

```sh
$ dotenvx-armor pull
```

Use a specific env file.

```sh
$ dotenvx-armor pull -f .env.production
```

Use a token or team.

```sh
$ dotenvx-armor pull --token token --team team
```
