# Armor Push

<GuideHeading lead={description} />

```sh
$ dotenvx-armor push
```

Use a specific env file.

```sh
$ dotenvx-armor push -f .env.production
```

Use a token or team.

```sh
$ dotenvx-armor push --token token --team team
```
