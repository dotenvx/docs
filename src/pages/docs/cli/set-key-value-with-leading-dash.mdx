# dotenvx set KEY -- "- + * ÷"

<GuideHeading lead={description} />

If your value starts with a dash (`-`), then place two dashes instructing the cli that there are no more flag arguments.

```sh
$ touch .env.ci

dotenvx set HELLO -f .env.ci -- "- + * ÷"
◈ encrypted HELLO (.env.ci)
```
