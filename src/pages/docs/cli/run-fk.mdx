# dotenvx run -fk

<GuideHeading lead={description} />

```sh
$ mkdir -p apps/app1
touch apps/app1/.env
dotenvx set HELLO world -fk .env.keys -f apps/app1/.env

dotenvx run -fk .env.keys -f apps/app1/.env -- yourcommand
```
