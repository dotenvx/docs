# VLT ⛨ Armor rotate

<GuideHeading lead={description} />

```sh
$ dotenvx-vlt armor rotate
```

Use a specific env file.

```sh
$ dotenvx-vlt armor rotate -f .env.production
```

Use a token.

```sh
$ dotenvx-vlt armor rotate --token token
```
