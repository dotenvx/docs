# dotenvx ext prebuild

<GuideHeading lead={description} />

Add it to your Dockerfile.

```sh
# Dockerfile
RUN curl -fsS https://dotenvx.sh | sh

...

RUN dotenvx ext prebuild
CMD ["dotenvx", "run", "--", "node", "index.js"]
```
