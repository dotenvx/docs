# dotenvx decrypt -k

<p className="lead">{description}</p>

```sh
echo "HELLO=World\nHOLA=Mundo" > .env
dotenvx encrypt
◈ encrypted (.env)
dotenvx decrypt -k HELLO
◇ decrypted (.env)
```

Even specify a glob pattern.

```sh
echo "HELLO=World\nHOLA=Mundo" > .env
dotenvx encrypt
◈ encrypted (.env)
dotenvx decrypt -k "HE*"
◇ decrypted (.env)
```
