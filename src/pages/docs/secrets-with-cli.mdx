# Encrypt secrets with Dotenvx CLI

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the Dotenvx CLI.

<CodeGroup>

<code title="Dotenvx" code="curl -sfS https://dotenvx.sh | sh">{`$ curl -sfS https://dotenvx.sh | sh
`}<span className="logger-success">◈ installed (dotenvx)</span></code>

<code title="with Ops ⛨" code="curl -sfS https://dotenvx.sh/ops | sh">{`$ curl -sfS https://dotenvx.sh/ops | sh
`}<span className="logger-success">◈ installed (dotenvx)</span>{'\n'}<span className="logger-success">⛨ installed (dotenvx-ops)</span></code>

</CodeGroup>

## 3. Encrypt

Encrypt your .env file.

<CodeGroup>

<code title="Dotenvx" code="dotenvx encrypt">{`$ dotenvx encrypt
`}<span className="logger-success">◈ encrypted (.env) + local key (.env.keys)</span></code>

<code title="with Ops ⛨" code="dotenvx encrypt">{`$ dotenvx encrypt
`}<span className="logger-success">◈ encrypted (.env) + armored key ⛨</span></code>

</CodeGroup>

## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```sh
dotenvx run -- yourcommand
```
