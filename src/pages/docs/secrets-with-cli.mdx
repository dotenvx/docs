# Encrypt secrets with Dotenvx CLI

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the Dotenvx CLI.

<InstallCodeGroup/>

## 2. Authenticate

Optionally, install dotenvx-ops and log in to your Dotenvx account. 

<InstallOpsCodeGroup/>

```sh
dotenvx ops login
```

## 3. Encrypt

Then encrypt your .env file.

```sh
dotenvx encrypt
```
