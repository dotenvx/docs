## 3. Encrypt

Encrypt your .env file.

<CodeGroup>

<code title="Dotenvx" code="dotenvx encrypt">{`$ dotenvx encrypt
`}<span className="logger-success">◈ encrypted (.env) + local key (.env.keys)</span></code>

`}<span className="logger-success">◈ encrypted (.env) + armored key ⛨</span></code>

## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```sh
dotenvx run -- yourcommand
```
