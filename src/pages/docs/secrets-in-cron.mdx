# Encrypt secrets in Cron

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Install dotenvx](/docs/install)
* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the dotenvx cli.

<CodeGroup>

```bash {{ title: "curl" }}
curl -sfS https://dotenvx.sh | sh
```

</CodeGroup>

## 2. Encrypt env file

Encrypt your `.env` file.

```sh {{ title: '$ terminal' }}
dotenvx encrypt
```

## 3. Inject secrets

Then inject your encrypted secrets in cron jobs with `dotenvx run`.

```sh {{ title: 'crontab' }}
# run every day at 8am
0 8 * * * dotenvx run -- /path/to/myscript.sh
```
