# Managing Access

## Click config icon

Visit the team page and click config next to your teammate's name.

<ScreenshotBrowser url="/assets/img/cloudinary/Screen_Shot_2022-10-11_at_10.57.30_PM_brcbuq.png" />

## Select environment

Select the environment to limit access to.

<ScreenshotBrowser url="/assets/img/cloudinary/Screen_Shot_2022-10-11_at_10.58.02_PM_rlpbwl.png" />

## Configure permissions

Configure your teammate's access.

<ScreenshotBrowser url="/assets/img/cloudinary/change-team-member-environment-permissions_tphkax.gif" />

* **View** - Permit your teammate to see values in the UI for this project
* **Push** - Permit your teammate to change values in the UI and from the CLI
* **Pull** - Permit your teammate to pull values using the CLI
