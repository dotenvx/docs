# Config

<GuideHeading lead={description} />

```ini
# .env
HELLO="World"
```
```js
// index.js
require('@dotenvx/dotenvx').config()
// or import '@dotenvx/dotenvx/config' if you're using esm

console.log(`Hello ${process.env.HELLO}`)
```
```sh
$ node index.js
Hello World
```
