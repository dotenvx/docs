## Curl

Our install.sh script is easiest. With curl:

```bash
curl -sfS https://dotenvx.sh/ops | sh
```

Or with wget:

```bash
wget -qO- https://dotenvx.sh/ops | sh
```

Read the [install.sh script here](https://dotenvx.sh/ops).

## Other

### Standalone

`dotenvx-ops` is a standalone binary, so (if you want) you can just download it directly:

```sh
# download it to `./dotenvx-ops`
curl -sfS --proto '=https' https://dotenvx.sh/ops/$(uname)/$(uname -m).tgz | tar xz

# install it to `/usr/local/bin/dotenvx-ops`
sudo install -m 755 dotenvx-ops /usr/local/bin

# check it works
dotenvx-ops help
```

Download a specific version:

```sh
# download version 0.6.9
curl -sfS --proto '=https' "https://dotenvx.sh/ops/$(uname)/$(uname -m).tgz?version=v0.6.9" | tar xz

# check the version
./dotenvx-ops --version
```

### Npm local

Use dotenvx-ops locally in your node project.

```sh
npm i @dotenvx/dotenvx-ops --save
```
```json
{
  "scripts": {
    "start": "./node_modules/.bin/dotenvx run -- node index.js"
  },
  "dependencies": {
    "@dotenvx/dotenvx": "^1.48.4",
    "@dotenvx/dotenvx-ops": "^0.14.1"
  }
}
```

### Npm global

You can also install globally using npm's `--global` flag.

```bash
npm install @dotenvx/dotenvx-ops --global
dotenvx-ops help
```

### Npx

Or prefer to run as needed, uses `npx`.

```bash
npx @dotenvx/dotenvx-ops help
```
