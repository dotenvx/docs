## Sync .env files

Go to [/new/project](https://ops.dotenvx.com/new/project) and [create your project](https://ops.dotenvx.com/new/project).

<ScreenshotBrowser url='https://github.com/user-attachments/assets/1e2be6ba-2734-40c6-9154-413ad374e097' www='ops.dotenvx.com' width='md:w-4/6' />

On the next screen you'll see a copy of your `.env.x` file.

<ScreenshotBrowser url='https://github.com/user-attachments/assets/2ada67da-dec1-42eb-8800-930ba1bf7619' www='ops.dotenvx.com' width='md:w-4/6' />

Add `.env.x` to your project's root.

```sh
# motdotla/your-project/.env.x
DOTENVX_PROJECT_ID=prj_664c683dddf155a6480d524aa7a5e42d

# 1. Add this .env.x file to your project's root
# 2. Commit it safely to code
# 3. Sync with dotenvx-ops sync
```

> The `DOTENVX_PROJECT_ID` uniquely identifies your project and lives inside your `.env.x` file.
