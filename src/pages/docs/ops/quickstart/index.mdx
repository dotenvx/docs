## Backup

> Back up your private key(s) from `.env.keys`.

Encrypt your `.env*` file (if you haven't already).

```bash {{ title: "Encrypt" }}
$ echo "HELLO=world" > .env
$ dotenvx encrypt
✔ encrypted (.env)
✔ key added to .env.keys (DOTENV_PRIVATE_KEY)
```

And back up your private key(s) inside `.env.keys` with `dotenvx-ops backup`.

<CodeGroup>

```bash {{ title: "Curl" }}
$ dotenvx-ops backup
✔ backed up [motdotla/your-project]
```

```bash {{ title: "Npm" }}
$ npx @dotenvx/dotenvx-ops backup
✔ backed up [motdotla/your-project]
```

</CodeGroup>

That's it! Your project's private key(s) have been backed up.
