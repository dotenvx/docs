## Getting Started

<CodeGroup>

```bash {{ title: "Curl" }}
# download it directly as a standalone binary
curl -sfS https://dotenvx.sh/ops | sh
dotenvx-ops login
dotenvx-ops backup
```

```bash {{ title: "Npm" }}
# install it with npm
npm install @dotenvx/dotenvx-ops --save
npx dotenvx-ops login
npx dotenvx-ops backup
```
