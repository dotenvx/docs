# Other Ways to Install

## Brew

Installing with [brew](https://brew.sh) is the most straight forward:

```bash
# install with Homebrew and then use the dotenvx command
brew install dotenvx/brew/dotenvx
```

## Shell

After brew our installer is easiest. With curl:

```bash
curl -fsS https://dotenvx.sh/ | sh
```

<Note>
  Wanna read that script before you run it? [github.com/dotenvx/dotenvx.sh/installer.sh](https://github.com/dotenvx/dotenvx.sh/blob/main/installer.sh)
</Note>

Or with wget:

```bash
wget -qO- https://dotenvx.sh/ | sh
```

## Npm

`dotenvx` is made with javascript â€“ so you can use also install with [npm](https://www.npmjs.com/package/@dotenvx/dotenvx).

With `npx`:

```bash
npx @dotenvx/dotenvx help
```

Globally:

```bash
npm install @dotenvx/dotenvx --global
dotenvx help
```

Locally:

```bash
npm i @dotenvx/dotenvx --save
```
```json
{
  "scripts": {
    "start": "./node_modules/.bin/dotenvx run -- node index.js"
  },
  "dependencies": {
    "@dotenvx/dotenvx": "^0.6.0"
  }
}
```

Or even in your code:

```js
// index.js
require('@dotenvx/dotenvx').config()

console.log(`Hello ${process.env.HELLO}`)
```

Find the [@dotenvx/dotenvx package here](https://www.npmjs.com/package/@dotenvx/dotenvx).

## Docker

We also provide a [docker image](https://hub.docker.com/r/dotenv/dotenvx) for your convenience:

```bash
docker run -it --rm -v $(pwd):/app dotenv/dotenvx help
docker run -it --rm -v $(pwd):/app dotenv/dotenvx run -- node index.js
```

Or in any image:

```
FROM node:latest
RUN echo "HELLO=World" > .env && echo "console.log('Hello ' + process.env.HELLO)" > index.js
RUN curl -fsS https://dotenvx.sh/ | sh
CMD ["dotenvx", "run", "--", "echo", "Hello $HELLO"]
```

## Standalone

`dotenvx` is a standalone binary, so (if you want) you can just download it directly:

```sh
# download it to `./dotenvx`
curl -fsS --proto '=https' https://dotenvx.sh/$(uname)/$(uname -m).tgz | tar xz

# install it to `/usr/local/bin/dotenvx`
sudo install -m 755 dotenvx /usr/local/bin

# check it works
dotenvx help
```

Download a specific version:

```sh
# download version 0.6.9
curl -fsS --proto '=https' "https://dotenvx.sh/$(uname)/$(uname -m).tgz?version=v0.6.9" | tar xz

# check the version
./dotenvx --version
```

<Note>
  You can view available versions at [github.com/dotenvx/releases](https://github.com/dotenvx/releases/tree/main)
</Note>

## GitHub Releases

You can also download straight from [GitHub Releases](https://github.com/dotenvx/dotenvx/releases).

<Note>
  If you download manually you'll need to move the binary somewhere in your PATH.
</Note>

## pkgx

[pkgx](https://pkgx.sh) is a single, standalone binary that can run anything. Since it can run anything, it can run `dotenvx`:

```sh
pkgx npx @dotenvx/dotenvx help
```
