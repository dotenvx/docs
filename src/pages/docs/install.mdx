## Brew

Installing with [brew](https://brew.sh) is the most straight forward:

```bash
# install with Homebrew and then use the dotenvx command
brew install dotenvx/brew/dotenvx
```

## Curl

Or download it directly as a standalone binary:

```bash
# download it to `/user/local/bin/dotenvx`
curl -fsS https://dotenvx.sh/ | sh

# check it works
dotenvx help
```

With wget:

```bash
# download it to `/user/local/bin/dotenvx`
wget -qO- https://dotenvx.sh/ | sh
```

You can view the [raw installer.sh script here](https://dotenvx.sh/installer.sh) before downloading.

## Npm

Or use [npm](https://www.npmjs.com/package/@dotenvx/dotenvx)'s global install:

```bash
# install globally with npm
npm install @dotenvx/dotenvx --global
```

Or install it locally, so your repo has it built-in for your teammates:

```bash
npm i @dotenvx/dotenvx --save
```
```json
{
  "scripts": {
    "start": "./node_modules/.bin/dotenvx run -- node index.js"
  },
  "dependencies": {
    "@dotenvx/dotenvx": "^0.6.0"
  }
}
```

## Docker

Use [docker](https://hub.docker.com/r/dotenv/dotenvx) if you prefer:

```bash
# use Docker if you prefer
docker run -it --rm -v $(pwd):/app dotenv/dotenvx help
docker run -it --rm -v $(pwd):/app dotenv/dotenvx run -- node index.js
```

Or in any image:

```
FROM node:latest
RUN echo "HELLO=World" > .env && echo "console.log('Hello ' + process.env.HELLO)" > index.js
RUN curl -fsS https://dotenvx.sh/ | sh
CMD ["dotenvx", "run", "--", "echo", "Hello $HELLO"]
```
