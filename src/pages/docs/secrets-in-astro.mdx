# Encrypt secrets in Astro

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Install dotenvx](/docs/install)
* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the Dotenvx Node.js SDK.

<InstallNodeSdkCodeGroup/>

## 2. Encrypt env file

Encrypt your .env file.

```sh {{ title: '$ terminal' }}
npx dotenvx encrypt
```

## 3. Inject secrets

Preface Astro scripts with `dotenvx run --` and read your env values in Astro.

```json {{ title: 'package.json' }}
{
  "scripts": {
    "dev": "dotenvx run -- astro dev",
    "build": "dotenvx run -- astro build",
    "preview": "dotenvx run -- astro preview"
  }
}
```

```astro {{ title: 'src/pages/api.js' }}
export async function GET() {
  return new Response(
    JSON.stringify({
      HELLO: process.env.HELLO,
    }),
    {
      status: 200,
      headers: {
        "Content-Type": "application/json",
      },
    }
  );
}
```
