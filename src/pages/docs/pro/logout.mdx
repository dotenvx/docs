# Log out 
<GuideHeading lead={description} />

## CLI 📟

**Log out**–from the cli.

```bash
$ dotenvx pro logout
✔ logged out [motdotla] from this device and revoked token [dxo_5ZrwRXV…]
⮕ next visit [https://pro.dotenvx.com/settings/devices] to view your devices
```

Optionally, you can view the revoked device from the web ui.

<ScreenshotBrowser url='https://github.com/user-attachments/assets/4c2e3c53-914a-4cdb-9a0a-ea8fbeab64ca' www='pro.dotenvx.com' width='md:w-4/6' />

That's it. Thanks for using dotenvx pro. 🏆

## Web 🖥️

Logging out from the web is a separate process.

**Click log out** under your profile picture at the top right (or go directly to [pro.dotenvx.com/logout](https://pro.dotenvx.com/logout)).
