# Encrypt secrets in Next.js

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the Dotenvx Node.js SDK and [@dotenvx/next-env](https://www.npmjs.com/package/@dotenvx/next-env).

<InstallNextSdkCodeGroup/>

Override @next/env in your package.json.

```json {{ title: 'package.json' }}
{
  ...
  "overrides": {
    "@next/env": "npm:@dotenvx/next-env"
  }
}
```

## 2. Encrypt

Encrypt your .env file.

<EncryptWithArmorNpxCodeGroup/>

## 3. Inject

Your encrypted secrets are automatically injected and readable in Next.js.

```tsx {{ title: 'src/app/api/route.tsx' }}
import { NextResponse } from 'next/server'

export async function GET() {
  return NextResponse.json({
    HELLO: process.env.HELLO,
  })
}
```

### Troubleshooting

Not taking effect? NPM can be inconsistent with overrides. Delete `package-lock.json`, `node_modules`, and reinstall.

```sh
rm -rf node_modules package-lock.json
npm install
```
