# Encrypt secrets in Clojure

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Create your account](https://dotenvx.com/signup) (optional)

## 1. Install

Get the Dotenvx CLI.

<InstallCodeGroup/>

## 2. Encrypt env file

Encrypt your `.env` file.

```sh
dotenvx encrypt
```

## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```clojure {{ title: 'index.clj' }}
(println "Hello" (System/getenv "HELLO"))
```

```sh
dotenvx run -- clojure -M index.clj
```
