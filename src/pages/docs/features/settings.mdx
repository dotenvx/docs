## Usage

Run `dotenvx settings`.

```bash
$ dotenvx settings --pretty-print
{
  "DOTENVX_SETTINGS_FILEPATH": "/Users/user/Library/Preferences/dotenvx/.env",
  "DOTENVX_HOSTNAME": "https://hub.dotenvx.com",
  "DOTENVX_LATEST_VERSION_LAST_CHECKED": "1712528133048"
}
```
