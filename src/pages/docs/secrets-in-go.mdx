## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run`.

```go {{ title: 'main.go' }}
package main

import (
  "fmt"
  "os"
)

func main() {
  fmt.Printf("HELLO: %s\n", os.Getenv("HELLO"))
}
```

```sh
dotenvx run -- go run main.go
```
