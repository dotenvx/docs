## Log in

> Log in to harden your keys.

<CodeGroup>

```bash {{ title: "Curl" }}
$ dotenvx-armor login
◈ logged in
```

```bash {{ title: "Npm" }}
$ npx @dotenvx/dotenvx-armor login
◈ logged in
```

</CodeGroup>

Encrypt your `.env*` file (if you haven't already).

```bash {{ title: "Encrypt" }}
$ echo "HELLO=world" > .env
$ dotenvx encrypt
◈ encrypted (.env) · armored ⛨
```

That's it! Your private key(s) are armored off machine.
