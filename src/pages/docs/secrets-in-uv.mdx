# Encrypt secrets in uv

<GuideHeading lead={description} />

## Prerequisites

To get the most out of this guide, you'll need to:

* [Install uv](https://github.com/astral-sh/uv#installation)
* [Create your account](https://dotenvx.com/signup) (optional)

## 0. Setup

Create uv project.

```sh
uv init hello-world
cd hello-world
```

## 1. Install

Get the Dotenvx CLI.

## 3. Inject

Then inject your encrypted secrets at runtime with `dotenvx run` into a uv project.

```python {{ title: 'main.py' }}
import os

def main():
    print(f"Hello {os.getenv('HELLO')}")


if __name__ == "__main__":
    main()
```

```sh
dotenvx run -- uv run main.py
```
